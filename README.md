# wx-English-Training
### 单词拼拼
#### 在微信小程序里拼写单词

* 自定义.txt单词表格式与Bash-English-Training和C-English-Training的词表部分相同，左侧单词，中间一个或数个\t制表符，右边中文。
* 还没有写查询本地.txt词表例句和释义的功能，暂时使用了一个简单的服务器API代替。。词库不全
* 内置9000左右单词量的词表，支持中译英 英译中 中英互译 ，顺序 倒序 乱序。支持简单的发音，带有通关模式。


* 若要加载本地自定义词表，请将编辑好的.txt发送到微信任意联系人或文件传输助手，再在小程序中点自定义，从聊天记录中加载。
* 想直接使用请在微信中搜素 单词拼拼
